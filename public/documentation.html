the title for the api is movie_api

this api will allow you to return almost anything you want about a given movie IE description, actors movie lists etc. It will be able to load all of this data to the server asyncronously

Buisness Logic URL HTTP Method Request body data format Response body data format
See all movies /movies GET a text object stating 'all movies'
see a movie by name /movies/:name GET a text object coming back as 'certain movie name'
see a certain movie director /movies/:name/:director GET a text object coming back as 'certain movie director'
see a certain genre of movie by name /movies/:name/:director/:genre GET a text object coming back as 'certain movie genre'
adds a new movie to the list of movies /movies POST a object added in text called 'added movie'
deletes a movie off the list of movies by their id /movies/:id DELETE a object deleted by their id
