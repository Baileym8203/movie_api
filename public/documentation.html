the title for the api is movie_api

this api will allow you to return almost anything you want about a given movie IE description, actors movie lists etc. It will be able to load all of this data to the server asyncronously

Buisness Logic URL HTTP Method Request body data format Response body data format
See all movies /movies GET a text object stating 'all movies'
see a movie by name /movies/:title GET a text object coming back as 'certain movie name'
see a certain movie director /movies/director/:director GET a text object coming back as 'certain movie director'
see a certain genre of movie by name /genre/:name GET a text object coming back as "Successful GET request to get data about a single genre by name"
adds a new movie to the lsit of favorites of the user /users/:id/movies/:movies_id PUT any text will send a text message "Successful PUT request added movies to the list of favorites"
deletes a new movie to the lsit of favorites of the user /users/:id/movies/:movies_id DELETE will send a text message "Successful DELETE request added movies to the list of favorites"
registers a new user /users POST will send a text message "Successful POST request to register new user"
deletes request to register new user /users DELETE will send a text message "Successful DELETE request to register new user"
shows directors name /director/:name GET will send a text message "Successful GET request to get data about a single director by name"
