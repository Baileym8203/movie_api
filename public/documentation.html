the title for the api is movie_api

this api will allow you to return almost anything you want about a given movie IE description, actors movie lists etc. It will be able to load all of this data to the server asyncronously

Business Logic URL HTTP Method Request body data format Response body data format
See all movies /movies GET a json formatted response showing all the movies { id_ -------------, Title: '-----------', description: '----------', Genre: { Name: '---------', Description: '----------' }, Director: { Name: '--------', Bio: '-------', Birth: '--------', Death: '------' }, ImagePath: '-------', Featured: '-------' }
see a movie by name /movies/:title GET a json formatted response showing all the movies with a certain title as shown { id_ -------------, Title: '-----------', description: '----------', Genre: { Name: '---------', Description: '----------' }, Director: { Name: '--------', Bio: '-------', Birth: '--------', Death: '------' }, ImagePath: '-------', Featured: '-------' }
see a certain movie director /movies/director/:director GET a json formatted response showing all the movies with a certain director as shown { id_ -------------, Title: '-----------', description: '----------', Genre: { Name: '---------', Description: '----------' }, Director: { Name: '--------', Bio: '-------', Birth: '--------', Death: '------' }, ImagePath: '-------', Featured: '-------' }
see a certain genre of movie by name /genre/:name GET a json formatted response showing all the movies with a certain Genre as shown { id_ -------------, Title: '-----------', description: '----------', Genre: { Name: '---------', Description: '----------' }, Director: { Name: '--------', Bio: '-------', Birth: '--------', Death: '------' }, ImagePath: '-------', Featured: '-------' }
adds a new movie to the list of favorites of the user /users/:Username/movies/:movies_id PUT in json format { FavoriteMovies: ['------------'] } a json formatted response showing the new user will come back { id_ -------------, Username: '---------', Password: '-------', Email: '-------', Birthday: '-------------', FavoriteMovies: ['------------'] }
deletes a favorite movie from the users favorite movies /users/:Username/movies/:movies_id DELETE will send a message in text as shown '------------' has been deleted!
registers a new user /users POST { id_ -------------, Username: '---------', Password: '-------', Email: '-------', Birthday: '-------------', FavoriteMovies: ['------------'] } will send a message in formatted json like shown { id_ -------------, Username: '---------', Password: '-------', Email: '-------', Birthday: '-------------', FavoriteMovies: ['------------'] } showing the input you PUT to be sent to the database
deletes request to register new user /users/:Username DELETE Username: '---------' was deleted
gets the director based on name /director/:name GET a json formatted response showing the director based on name { id_ -------------, Title: '-----------', description: '----------', Genre: { Name: '---------', Description: '----------' }, Director: { Name: '--------', Bio: '-------', Birth: '--------', Death: '------' }, ImagePath: '-------', Featured: '-------' }
